BASIC (WEB PREZENTACIJA)

Basic paket predstavlja izradu klasičnih web prezentacionih statičkih sajtova, pogodnih za prezentovanje svih informacija koje bi zanimale ljude koje ovim putem targetirate na internetu. Web prezentacioni sajtovi predstavljaju brz i efikasan način da izreklamirate sebe i svoje usluge. Odabirom ovog paketa, dobijate web sajt unikatnog dizajna koji je prilagodljiv svim vrstama smart urađaja, integraciju mape, galerije slika i drugo... U okviru Basic paketa, održavanje web stranice Vam je osigurano do 6 meseci.

STANDARD (WEB SAJT SA CMS FUNKCIONALNOSTIMA)

Standard paket predstavlja izradu web sajtova dinamickog tipa sa cms funkcionalnostima. U okviru Standard paketa dobijate mogucnost upotrebe baze podataka i sistema koji samostalno moze uticati na nju. Odabirom ovog paketa dobijate mogucnost integraciju mape, galerije, izradu cms sistema preko koga mozete samostalno azurirati sajt, obradu i pripremu grafickog materijala i drugo..
